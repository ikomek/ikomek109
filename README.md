**Политика конфиденциальности**

Политика конфиденциальности персональной информации (далее — Политика) действует в

отношении всей информации и личных данных Пользователя во время использования

мобильного приложения iKOMEK 109.

Согласие Пользователя с Политикой, выраженное им распространяется на весь период

использования мобильного приложения iKOMEK 109.

Использование мобильного приложения iKOMEK 109 означает безоговорочное согласие

Пользователя с настоящей Политикой и указанными в ней условиями обработки его

персональной информации; в случае несогласия с этими условиями Пользователь должен

воздержаться от использования мобильного приложения iKOMEK 109.

1\. Персональная информация Пользователей, которую обрабатывает мобильное

приложенное

iKOMEK 109

1\.1. В рамках настоящей Политики под «персональной информацией Пользователя»

понимаются:

1\.1.1. Персональная информация, которую Пользователь предоставляет о себе

самостоятельно при регистрации или в процессе использования мобильного приложения

iKOMEK 109, включая персональные данные Пользователя. Обязательная для использования

мобильного приложения iKOMEK 109 информация помечена специальным образом. Иная

информация предоставляется Пользователем на его усмотрение.

1\.1.2. Данные, которые автоматически передаются мобильным приложением iKOMEK 109 в

процессе его использования с помощью установленного на устройстве Пользователя

программного обеспечения, в том числе IP-адрес, данные файлов cookie, информация о

браузере и данных геологической локации Пользователя (или иной программе, с помощью

которой осуществляется доступ к мобильному приложению iKOMEK 109), технические

характеристики оборудования и программного обеспечения, используемых Пользователем,

дата и время доступа к мобильному приложению iKOMEK 109 и иная подобная информация.

1\.1.3. Иная информация о Пользователе, обработка которой предусмотрена условиями

использования отдельных сервисов мобильного приложения iKOMEK 109.

1\.2. Настоящая Политика применима только к информации, обрабатываемой в ходе

использования мобильного приложения iKOMEK 109.

1\.3. Мобильное приложение iKOMEK 109 не проверяет достоверность персональной

информации, предоставляемой Пользователем, и не имеет возможности оценивать его

дееспособность. Однако мобильное приложение iKOMEK 109 исходит из того, что

пользователь предоставляет достоверную и достаточную персональную информацию и

поддерживает эту информацию в актуальном состоянии.

2\. Цели обработки персональной информации Пользователей

2\.1. Мобильное приложение iKOMEK 109 собирает и хранит только ту персональную

информацию, которая необходима для использования мобильного приложения iKOMEK 109,

за исключением случаев, когда законодательством предусмотрено обязательное хранение

персональной информации в течение определенного законом срока.

2\.2. Персональную информацию Пользователя обрабатывает в следующих целях:



<a name="br2"></a> 

2\.2.1. Идентификация Пользователя в рамках авторизации Пользователя в мобильном

приложении iKOMEK 109;

2\.2.2. Предоставление Пользователю сервисов мобильного приложения iKOMEK 109;

2\.2.3. Связь с Пользователем, в том числе направление уведомлений, запросов и

информации, касающихся использования мобильного приложения iKOMEK 109, а также

обработка запросов и обращений от Пользователя;

2\.2.4. Улучшение мобильного приложения iKOMEK 109, удобства их использования,

разработка новых сервисов;

2\.2.5. Проведение статистических и иных исследований, на основе обезличенных данных.

3\. Условия обработки персональной информации Пользователей и её передачи третьим

лицам

3\.1. Мобильное приложение iKOMEK 109 хранит персональную информацию Пользователей в

соответствии с внутренними регламентами конкретных сервисов.

3\.2. В отношении персональной информации Пользователя сохраняется ее

конфиденциальность, кроме случаев добровольного предоставления Пользователем

информации о себе для общего доступа неограниченному кругу лиц. При использовании

отдельных сервисов, Пользователь соглашается с тем, что определённая часть его

персональной информации становится общедоступной.

3\.3. Мобильное приложение iKOMEK 109 вправе передать персональную информацию

Пользователя третьим лицам в следующих случаях:

3\.3.1. Пользователь выразил согласие на такие действия, использование мобильного

приложения iKOMEK 109 принимать за согласие Пользователя на условия Политики;

3\.3.2. Передача необходима для использования Пользователем сервисов мобильного

приложения iKOMEK 109;

3\.3.3. В целях обеспечения возможности защиты прав и законных интересов мобильного

приложения iKOMEK 109 или третьих лиц в случаях, когда Пользователь нарушает

настоящую Политику, либо документы, содержащие условия использования конкретных

сервисов мобильного приложения iKOMEK 109.

3\.3.4. В результате обработки персональной информации Пользователя путем ее

обезличивания получены обезличенные статистические данные, которые передаются

третьему лицу для проведения исследований, выполнения работ или оказания услуг по

использованию мобильного приложению iKOMEK 109.

3\.4. При обработке персональных данных Пользователей мобильного приложению iKOMEK

109 руководствуется Законом Республики Казахстан «О персональных данных и их защите».

4\. Изменение и удаление персональной информации. Обязательное хранение данных

4\.1. Пользователь может в любой момент изменить (обновить, дополнить) предоставленную

им персональную информацию или её часть, воспользовавшись функцией редактирования

персональных данных в разделе «Данные Пользователя».

4\.2. Права, предусмотренные пп. 4.1. настоящей Политики могут быть ограничены в

соответствии с требованиями законодательства.



<a name="br3"></a> 

5\. Меры, применяемые для защиты персональной информации Пользователя

5\.1. Сервисом мобильного приложения iKOMEK 109 принимаются необходимые и

достаточные организационные и технические меры для защиты персональной информации

Пользователя от неправомерного или случайного доступа, уничтожения, изменения,

блокирования, копирования, распространения, а также от иных неправомерных действий с

ней третьих лиц.

6\. Изменение Политики конфиденциальности. Применимое законодательство

6\.1. Сервис мобильного приложению iKOMEK 109 имеет право вносить изменения в

настоящую Политику конфиденциальности. При внесении изменений в актуальной редакции

указывается дата последнего обновления. Новая редакция Политики вступает в силу с

момента ее размещения, если иное не предусмотрено новой редакцией Политики.

6\.2. К настоящей Политике и отношениям между Пользователем и мобильным приложением

iKOMEK 109, возникающим в связи с применением данной Политики конфиденциальности.

7\. Обратная связь. Вопросы и предложения

7\.1. Все предложения или вопросы по поводу настоящей Политики Пользователь вправе

направлять в Службу поддержки приложения мобильного приложения iKOMEK 109 по
